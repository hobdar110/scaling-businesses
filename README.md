FluidicDOM Architecture Solutions

A clean, responsive, and semantically structured 3-page web platform engineered during my development internship at **Decode Labs**. 

This project serves as a practical implementation of dynamic UI layout design, replacing rigid, device-specific assumptions with a content-driven, fluid layout engine that adapts beautifully across all viewport sizes.

 🚀 Key Frameworks & Logic
The Art of Fluidity:** Built with a mobile-first mindset, ensuring the interface handles content organically from small handheld screens up to large desktop monitors.
  Semantic DOM Architecture:** Prioritizes strict structural markup integrity and accessibility over heavy, bloated UI frameworks.
  Bridges, Not Fences:** Uses content-driven media query breakpoints (`@media (min-width: 768px)`) to apply graceful layout upgrades cleanly exactly where viewport space permits.

 🛠️ Project Structure
The repository consists of a modular layout separation system:
 `index.html` – The core corporate platform showcasing structural grid concepts.
 `about.html` – A detailed breakdown of the workflow engine and box-model design stages.
 `contact.html` – An asymmetric grid interface incorporating functional form controls and mock data indicators.
 `project2.css` – The centralized design system featuring global CSS variables, custom typography constraints, and responsive alignments.

💻 Tech Stack
HTML5 – Semantic element tagging and accessible layout design.
CSS3 – Custom variables (`:root`), Flexbox layout tools, and CSS Grid architectures.
